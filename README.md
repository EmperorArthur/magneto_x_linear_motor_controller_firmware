# Peopoly Magneto X Linear Motor Control Board Firmware

按键功能:

Disable: 失能两个电机
Enable: 清除错误，和重新使能两个电机

LED灯效:

X轴: 当有错误时，灯是红色，正常时，灯是绿色
Y轴: 当有错误时，灯是红色，正常时，灯是绿色
